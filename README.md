# Naive Bayes for Text Classification

This project implements Naive Bayes for text classification using the 20 Newsgroups dataset. The model is trained to categorize documents into one of the 20 newsgroups based on their content. The algorithm achieves 75%/52% accuracy on the train/test data.

For more on Naive Bayes models and this implementaion, check out the [blog](https://paulkroe.github.io/posts/naive-bayes/).
